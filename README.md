# node-module

A very simple node package that allows you to merge sort a list.

## Features

* Merge Sort

## Installation

```bash
npm i maghribnode

```

## Usage

```bash
  const { merge_sort } = require('maghribnode');

  let items = [3, 10, 1, 0, 28, 17, 8, 100]
  let result = merge_sort(items)
  console.log(result) # [0, 1, 3, 8, 10, 17, 28, 100]

```
